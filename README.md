# vim-snazzy
> An elegant Vim theme with bright (dare I say snazzy?) colors

## Installation

Add `colorscheme snazzy` to your `~/.vimrc`.

### vim-plug
Add the following to your `~/.vimrc` file and run `PlugInstall`.

    Plug 'angamaiton/vim-snazzy'
